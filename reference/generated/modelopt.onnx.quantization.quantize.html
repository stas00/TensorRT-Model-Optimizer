Quantize the given onnx model.

Parameters:

Example: [‚ÄòConv‚Äô] indicates that only ops of type ‚ÄòConv‚Äô should be quantized. If this list is None (default), all supported operators are quantized. This flag does not support regular expression.

  • use_external_data_format (bool) ‚Äì If not None, this path will be used to store the weights of the quantized model.

  • keep_intermediate_files (bool) ‚Äì

    If False, only save the converted ONNX files for the user. Otherwise, keep all intermediate files

    generated during the ONNX models‚Äô conversion/calibration.

  • output_path (str) ‚Äì Output filename to save the converted ONNX model. If None, save in the same directory as the original ONNX model with .quant suffix.
